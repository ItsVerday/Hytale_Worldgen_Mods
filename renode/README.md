# Renode
Renode is a Hytale Plugin to automatically add support for custom nodes in Hytale's Asset Node Editor.
The plugin accomplishes this by automatically generating configuration files for these nodes for the Asset Node Editor to pull from.
Using Renode, developers can easily add custom Node Editor nodes to their plugins using a simple, flexible API, instead of having to dig into the specifics of how Hytale's Node Editor configs work.

Renode is intended to be used by plugin developers and modders using content added by these plugins.
End users of these mods do not need to install Renode.
